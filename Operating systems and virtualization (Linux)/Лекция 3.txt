Урок 3. Права доступа и пользователи
Пользователи:
- суперпользователи (root)
- обычный пользователь (user)
- системный пользователь (deamon)
/etc/passwd - список пользователей
/etc/group - группы пользователей
/etc/shadow - пароли пользователей
cat file_name - просмотр файла
sudo cat file_name - просмотр файла с правами администратора
Все операции надо пользователями должны осуществляться с правами администратора
Добавить пользователя
sudo useradd -s /bin/bash -m -d /home/testuser testuser
sudo adduser testuser2
Измнить пароль
sudo passwd testuser
